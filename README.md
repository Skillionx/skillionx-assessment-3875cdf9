# Eenvoudige Klantenbeheer Applicatie

Ontwikkel een eenvoudige consoleapplicatie in C# die basisfunctionaliteit biedt voor het beheren van klanten. De applicatie moet de volgende functies ondersteunen:

1.  **Klant Toevoegen**: Gebruikers moeten nieuwe klanten kunnen toevoegen met een unieke ID, naam en e-mailadres.
2.  **Klanten Weergeven**: Alle toegevoegde klanten moeten in een lijst kunnen worden weergegeven.
3.  **Klant Zoeken**: Gebruikers moeten een klant kunnen zoeken op basis van hun ID.
4.  **Klant Verwijderen**: Gebruikers moeten een klant kunnen verwijderen op basis van hun ID.

Gebruik de principes van objectgeoriënteerd programmeren en zorg voor duidelijke code met commentaar. De data hoeft niet persistent opgeslagen te worden; een in-memory collectie is voldoende voor deze opdracht.

## Opdracht

Implementeer de gevraagde functionaliteit.
